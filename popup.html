This extension converts Mobalytics paragon board guides into accessible tables for screen reader users.

Navigate to a Diablo 4 build guide on Mobalytics to see it in action.

Screen Reader Navigation

  • Use T to jump between tables
  • Use Ctrl+Alt+Arrow to navigate cells
  • Selected nodes are marked with "selected" text
